What Did I Learn in ICS4U

  • JavaScript
    • After Adobe and C++, JavaScript is the next level after the two, combining features from each program together into JavaScript. I learned JavaScript well through working on the summatives and the modules prepared on this website. Using previous knowledge and the skills we learned this year, I learned quite a lot about JavaScript and how to run it properly on Khan Academy, Glitch and p5.js.
  • HTML/CSS
    • I started learning HTML through Khan Academy and Free Code Camp and learned multiple things about setting up a webpage (with doctype and opening and closing code) and using elements to create a suffient website. I used a lot of HTML on this website to get it together. As for CSS, I approached it with the previous knowledge of HTML and used it on some factors of my website (from the gradient background to the coloured text). I would say I learned HTML/CSS very well throughout the course.
  • Glitch/p5.js
    • When the hackathon was assigned, my group turned towards Glitch, a free coding website and coded a complex program (Study Bunny) using HTML, CSS and JavaScript. Glitch made learning how to combine the three easier to comprehend. As for p5.js, which we used for the coding train summative, was also easy to use for beginners. It was separated into bits (one js, one html, one css file) and it was preset so you could work around it. Using Glitch and p5.js editor worked really well in expanding my understanding in those three programs.
